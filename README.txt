This is a schedule manager project. Currently, the following features are implemented: 
Sign in and out
Multiple users 
Change settings (Skip login and switch theme, only availble to admin users) 
Create new tasks (Set task name, task description, and due date) 
View tasks (Will only show anything for users who have created databases, wont show anything otherwise)
